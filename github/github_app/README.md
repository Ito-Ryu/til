# Github Acions
## 概要
GitHub の機能を拡張するツール

## 特徴
### 1. イベントベースの自動化
IssueやPull Requestの更新などのアクションに反応して動く。

### 2. 詳細なアクセス許可
必要なリポジトリや機能のみにアクセス制限が可能。

### 3. 安全なトークン
1時間で期限が切れるトークンを使用するため、セキュリティリスクが低い。

### 4. ボット機能
ユーザー個人に紐付かないため、特定の人が退職しても連携が止まらない。

### 5. 連携事例
Slackへの通知、CI/CDツール（GitHub Actionsなど）、自動レビューbotなど。
